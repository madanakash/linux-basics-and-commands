# What is Linux?

Linux is an **open-source operating system kernel** developed by Linus Torvalds in 1991.  
A *kernel* is the core part of an OS â€” it manages hardware, memory, processes, and system calls.

Linux by itself is only a kernel.  
To form a complete operating system, it is combined with:
- System utilities  
- Shells  
- Package managers  
- Libraries  
- Desktop environments (optional)

These complete systems are called **Linux distributions** (e.g., Ubuntu, Debian, CentOS).
